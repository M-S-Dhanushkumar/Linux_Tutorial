#Linux Commands

## uname
***********

``````
uname - print system information
``````
To print uname without options

`````
uname
`````
To print all information

```````
uname -a

````````
To print the kernel name

````
$ uname -s

````````
To print the kernel release

``````
$ uname -r

``````
