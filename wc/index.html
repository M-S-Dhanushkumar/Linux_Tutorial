wc

wc - print newline, word, and byte counts for each file wc without options will display (number of lines),(number of words) and (number of bytes) of the file

wc file.txt

To Count Number of Lines

wc -l file.txt

To Display Number of Words

wc -w file.txt

To Count Number of Bytes and Characters

wc -c file.txt
wc -m file.txt
