### 26. df
df - report file system disk space usage
```
df
```
To display all the file system
```
df -a
```
To display size in human readable format
```
df -h /home/
```
To get complete grand total
```
df -h --total
```
To display file type
```
df -T /home/ilugc
```
To display disk space usage of current dir
```
df -Th
```
